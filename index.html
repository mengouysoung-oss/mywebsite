About Cryptography

What is Cryptography?

Cryptography is the practice and study of techniques for secure communication in the presence of adversaries. It involves transforming readable information (plaintext) into an unreadable format (ciphertext) using mathematical algorithms and keys.

Types of Cryptography

  • Symmetric Cryptography: Uses the same key for encryption and decryption
  • Asymmetric Cryptography: Uses a public key for encryption and a private key for decryption
  • Hash Functions: One-way functions that produce fixed-size outputs
